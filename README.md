# lxqt-themes-xfce
LXQt ports of old-school (GTK2) Xfce themes

As of 27 Aug. 2023, this repository contains ports of *all* the old GTK2 themes aside from Flat, which frankly was both ugly and redundant when Dusk exists. Previews below:
