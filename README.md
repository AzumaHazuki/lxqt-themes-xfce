# lxqt-themes-xfce
LXQt ports of old-school (GTK2) Xfce themes

As of 18 Feb. 2023, this repository contains Xfce-4.0, Xfce-4.2, and Xfce-4.4 themes, as well as palettes and Openbox themes which do their best within the limits of Openbox's theme specification to mimic the respective Xfwm4 window borders. Openbox does not support rounded corners and some other features, but these are still rather close.
